# Owner Notification Template

Usar con `sessions_send` hacia la sesión privada de Alejo.

## Formato sugerido

Nuevo ticket {{ticket_id}}
- Remitente: {{from_name}} ({{from_id}})
- Tipo: {{type}}
- Prioridad: {{priority}}
- Resumen: {{summary}}
- Acción sugerida: {{next_action}}

Si es una cita, añadir:
- Fecha/hora propuesta: {{requested_time}}

Si es urgente, abrir con:
- URGENTE: requiere revisión pronta.
